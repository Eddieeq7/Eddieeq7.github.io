# Skill Cube Textures

Place your programming language logo images in this folder with the following exact names:

## Required Files:
- `cpp.png` - C++ logo
- `python.png` - Python logo
- `javascript.png` - JavaScript logo
- `typescript.png` - TypeScript logo
- `node.png` - Node.js logo
- `react.png` - React logo

## Image Requirements:
- **Format**: PNG with transparent background
- **Size**: 512x512px or 1024x1024px (square)
- **Quality**: High resolution for crisp rendering
- **Background**: Transparent or dark (matches #0a0a0c)
- **Logo**: Centered, well-contrasted

## Download Sources:

### Official Logos:
- **C++**: https://raw.githubusercontent.com/isocpp/logos/master/cpp_logo.png

### Alternative Sources:
- Simple Icons: https://simpleicons.org/
- DevIcon: https://devicon.dev/
- Vector Logo Zone: https://www.vectorlogo.zone/

## Tips:
1. Use PNG-8 with transparency for smaller file sizes
2. Ensure logos are visible on dark backgrounds
3. White or light-colored logos work best
4. Keep file sizes under 200KB each for performance
5. Test that logos are centered in the image

## Fallback:
If images are not found, the cube will render with styled text labels and gradients.
